# Homebrew Tap for BuildGuard

This is the official [Homebrew](https://brew.sh) tap for [BuildGuard](https://github.com/Build-Flow-Labs/buildguard-oss).

## Install

```bash
brew tap Build-Flow-Labs/buildguard
brew install buildguard
```

## Upgrade

```bash
brew upgrade buildguard
```

## Verify

```bash
buildguard version
```

## About

BuildGuard is a compliance scanner for GitHub organizations. It evaluates repositories against 5 built-in security and governance policies and reports results in the terminal.

Learn more at [github.com/Build-Flow-Labs/buildguard-oss](https://github.com/Build-Flow-Labs/buildguard-oss).
